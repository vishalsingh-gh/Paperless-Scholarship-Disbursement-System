# 🎓 Paperless Scholarship Disbursement System

A full-stack web application designed to digitize and streamline the scholarship application process. This system eliminates paperwork by allowing students to submit applications and documents online, while administrators can verify and approve them efficiently.

---

## 🚀 Features

### 👨‍🎓 Student Module
- User Registration & Login
- Fill Personal Profile Details
- Upload Required Documents:
  - Aadhaar Card
  - Marksheet
  - Income Certificate
- Real-time Upload Status (Submitted ✅)
- Progress Tracking (3/3 Completed)
- File Preview Option
- Final Submission: **Forward for Scholarship**

---

### 👨‍💼 Admin Module
- Admin Login
- View All Student Applications
- View Uploaded Documents
- Approve / Reject Applications
- Track Application Status (Pending / Approved / Rejected / Forwarded)

---
